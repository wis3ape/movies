#Instructions

Install node & bower dependencies:
`npm i && bower install`

To run the project for development please run:
`grunt serve-dev`

To run it for production run instead:
`grunt serve-prod`

It's importante to notice that you might need to install grunt-cli globally:
`npm i -g grunt-cli`
